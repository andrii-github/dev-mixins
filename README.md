# Dev mixins for fast coding (SCSS) 

## Instructions
		bower install dev-mixins-scss

#### It contains mixins for:
1. position
2. centering of elements
3. size
4. margin
5. padding
6. padding & margin reset
7. clearfix
8. display inline without spaces
9. unstyled lists
10. border-radius
11. border-color
12. responsive image
13. background
14. overlay
15. font styling
16. scrollbar
17. selection
18. placeholders
